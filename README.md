# Forex Analytics App

Professional forex analytics dashboard with real-time charts, technical indicators, and trading signals.

## Features
- Real-time EURUSD candlestick charts
- Technical indicators (SMA, RSI, MACD)
- Automated buy/sell signals
- Responsive dashboard design

## Tech Stack
- Frontend: React, TypeScript, Tailwind CSS, Lightweight Charts
- Backend: Node.js, Express, Alpha Vantage API
- Deployment: Vercel + Render
